Conseils pour le jour J

  • Repetez avec un chrono. 5 minutes c'est court. Si vous depassez, coupez dans la slide 10 (Avantage) : 3 points au lieu de 6.
  • Adam, appuie-toi sur l'ecran. Tes deux slides (Produit + Traction) sont ultra-visuelles. Pointe les screenshots, suis la timeline du doigt. Tu n'as pas besoin de tout reciter — montre et commente.
  • Pas de "je passe la parole a...". Les transitions sont ecrites pour etre naturelles. Terminez votre phrase, le suivant enchaine.
  • Le coeur du pitch = slides 2-3-4. Probleme → Solution → Produit. Soyez les plus percutants la-dessus.
  • Nathan ouvre et donne le rythme. Les 30 premieres secondes definissent l'energie du pitch. Posture droite, voix posee, contact visuel avec le jury.
  • Louis ferme avec conviction. Le closing et l'ask sont les derniers mots que le jury entend. Finir fort.
  • Chaque chiffre doit etre dit a voix haute : 72%, 500k, 18 mois, 16 Mds, 633 commits... Les chiffres marquent les esprits.
  • Les PAUSE dans le script sont la pour respirer. Ne les supprimez pas — le silence rend le propos plus impactant.
  • Positionnement physique (suggestion) : Nathan a gauche (il commence), Juliette centre-gauche, Adam centre-droit, Louis a droite (il finit).
  • Questions du jury — Tech : Nathan. Business/Marche : Juliette ou Louis. Produit/UX : Adam. Finance/Pricing : Louis.
